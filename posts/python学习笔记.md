#Python学习笔记
##浅复制与深复制
* 浅复制

      >>> a = [1,2,[3,4]]
      >>> b = list(a)
      >>> b is a
	  False
      >>> b.append(100)
      >>> b
      [1, 2, [3, 4], 100]
      >>> a
      [1, 2, [3, 4]]
      >>> b[2][0] = -100
      >>> b
      [1, 2, [-100, 4], 100]
      >>> a
      [1, 2, [-100, 4]]
* 深复制

      >>> import copy
      >>> a = [1,2,[3,4]]
      >>> b = copy.deepcopy(a)
      >>> b[2][0] = -100
      >>> b
      [1, 2, [-100, 4]]
      >>> a
      [1, 2, [3, 4]]
